# 3D Gaussian Splatting for Real-Time Radiance Field Rendering 
提出了**3D高斯溅射（3D Gaussian Splatting）** 方法，首次实现了辐射场的**高质量实时渲染（≥30fps，1080p）**，同时保持与当前最快辐射场方法相当的训练时间，视觉质量媲美甚至超越当时的SOTA方法Mip-NeRF360。核心是将3D高斯作为场景表示基元，结合自适应密度优化与实时可微渲染器，融合了显式点基表示的快速光栅化优势和隐式辐射场的优化灵活性。

## 一、研究背景与问题
神经辐射场（NeRF）及衍生方法革新了新视角合成，但存在核心矛盾：
1. **高质量方法（如Mip-NeRF360）**：训练耗时极长（48h）、渲染速度慢（0.071fps），无法实时；
2. **快速方法（如InstantNGP、Plenoxels）**：训练/渲染快，但为速度牺牲了视觉质量，且难以实现1080p实时渲染；
3. 现有表示要么依赖神经网络/体素射线步进（计算昂贵、有噪声），要么依赖多视图立体（MVS）点云（存在过/欠重建伪影）。

论文的核心目标：**在不损失视觉质量的前提下，实现辐射场的实时渲染，且训练时间与快速方法持平**。

## 二、核心贡献
1. 提出**各向异性3D高斯**作为辐射场的高质量非结构化表示，无需MVS点云，仅通过SfM稀疏点云即可初始化；
2. 设计**3D高斯属性优化+自适应密度控制**的联合策略，实现场景的精准、紧凑表示；
3. 开发**基于瓦片的快速可微渲染器**，支持各向异性溅射和可见性感知的α混合，实现实时前向渲染和高效反向传播，无溅射数量硬限制。

## 三、方法整体框架
方法输入为**多视角图像+SfM相机标定得到的稀疏点云**，整体分为三部分：
1. **3D高斯初始化**：从SfM稀疏点云生成初始3D高斯集，定义位置、各向异性协方差、不透明度、球谐系数（颜色）；
2. **自适应优化**：迭代优化高斯属性，同时通过**克隆/分裂**实现高斯密度的自适应增删，修正过/欠重建；
3. **实时可微渲染**：将3D高斯投影为2D各向异性溅射，通过瓦片式光栅化、深度排序、α混合实现实时渲染，且支持梯度反向传播。

## 四、核心技术细节
### 4.1 可微3D高斯溅射（Differentiable 3D Gaussian Splatting）
#### 4.1.1 3D高斯的定义
3D高斯以SfM点为中心${\mu}$，由**满秩3D协方差矩阵${\Sigma}$** 定义，世界空间中的高斯分布为：
$$G({x}) = e^{-\frac{1}{2}({x}-{\mu})^T {\Sigma}^{-1} ({x}-{\mu})} \tag{4}$$
该高斯与不透明度$\alpha$相乘，参与后续的α混合过程。

#### 4.1.2 3D高斯到2D溅射的投影
渲染时需将3D高斯投影到图像空间为2D各向异性溅射，投影过程由**视图变换$W$**和**投影变换的雅可比矩阵$J$** 完成，相机坐标系下的协方差矩阵${\Sigma}'$为：
$${\Sigma}' = J W {\Sigma} W^T J^T \tag{5}$$
取${\Sigma}'$的前2×2子矩阵，即得到图像空间的2D协方差矩阵，对应2D各向异性高斯溅射，无需估计法线（解决了SfM点云法线缺失的问题）。

#### 4.1.3 协方差矩阵的可优化表示
协方差矩阵需满足**半正定**，直接优化易出现无效值，因此将其分解为**缩放矩阵$S$**和**旋转矩阵$R$**的组合：
$${\Sigma} = R S S^T R^T \tag{6}$$
- 缩放：用3D向量${s}$表示，通过指数激活保证非负；
- 旋转：用四元数${q}$表示，通过归一化保证为单位四元数，避免旋转矩阵的正交性约束。

该表示既保留了各向异性的表达能力，又让梯度下降优化成为可能，论文还推导了缩放/旋转对协方差矩阵的显式梯度（见附录A），避免自动微分的开销。

### 4.2 3D高斯的自适应密度控制优化
优化目标是调整3D高斯的**位置${p}$、不透明度$\alpha$、协方差${\Sigma}$、球谐系数（颜色${c}$）**，同时通过自适应增删高斯解决过/欠重建，分为**优化策略**和**密度控制**两部分。

#### 4.2.1 优化策略
1. **优化器与激活**：采用随机梯度下降（SGD），$\alpha$经sigmoid激活约束在$[0,1)$，缩放${s}$经指数激活保证非负；
2. **损失函数**：结合$L_1$损失和D-SSIM损失，平衡像素级误差和感知质量：
   $$\mathcal{L} = (1-\lambda)\mathcal{L}_1 + \lambda\mathcal{L}_{D-SSIM} \tag{7}$$
   实验中$\lambda=0.2$；
3. **球谐系数（SH）优化**：SH用于表示视角相关的颜色，为避免角度信息缺失导致的优化不稳定，先优化0阶SH（漫反射色），每1000次迭代增加1阶，直至4阶；
4. **初始协方差**：初始化为各向同性高斯，轴长为到最近3个SfM点的平均距离。

#### 4.2.2 自适应密度控制（克隆+分裂）
优化预热后，每100次迭代执行一次密度调整，**移除$\alpha<\epsilon_\alpha$的透明高斯**，并对**视图空间位置梯度大于$\tau_{pos}$**的高斯进行增密（梯度大表示区域未充分重建），分两种情况：
1. **欠重建（小高斯）**：**克隆**高斯，沿位置梯度方向移动副本，增加几何覆盖；
2. **过重建（大高斯）**：**分裂**高斯为两个，缩放因子除以$\phi=1.6$，基于原高斯的概率密度函数（PDF）采样新位置，细化几何细节。

此外，每3000次迭代将部分高斯的$\alpha$置0，通过优化重新激活有效高斯，抑制相机附近的无效浮点数高斯，控制总高斯数量（实验中场景高斯数为1-5百万）。

### 4.3 高斯的快速可微渲染器
设计**基于16×16瓦片的GPU光栅化器**，核心解决**实时渲染**和**无限制梯度传播**问题，分为**前向渲染**和**反向传播**两部分，整体流程见Algorithm 2。

#### 4.3.1 前向渲染（实时光栅化）
1. **视锥剔除**：保留99%置信区间与视锥相交的高斯，通过保护带剔除近平面/视锥外的不稳定高斯；
2. **瓦片分配**：将屏幕划分为16×16瓦片，为每个高斯分配其重叠的所有瓦片，生成**瓦片ID+视图空间深度**的64位排序键；
3. **GPU基数排序**：基于排序键对高斯全局排序，实现**瓦片内的深度从远到近排序**，无需逐像素排序，大幅提升效率；
4. **瓦片级α混合**：为每个瓦片启动线程块，将高斯加载到共享内存，逐像素从远到近混合颜色和不透明度，当像素不透明度饱和（$\alpha\to1$）时停止计算，最大化并行效率。

该渲染器的α混合是**近似的**，但当溅射接近像素大小时，近似误差可忽略，且无视觉伪影。

#### 4.3.2 反向传播（无溅射数量限制）
与现有方法（如Pulsar）限制梯度传播的溅射数量不同，该方法支持**任意数量高斯的梯度回传**，核心策略：
1. **反向遍历**：复用前向的排序结果，从近到远遍历瓦片内的高斯；
2. **中间不透明度恢复**：不存储逐高斯的中间不透明度，而是通过**前向最终累积不透明度**除以各高斯的$\alpha$，恢复梯度计算所需的系数；
3. **数值稳定性**：跳过$\alpha<1/255$的高斯，将$\alpha$上界钳位为0.99，混合时避免不透明度超过0.9999，防止除零/数值爆炸。

### 4.4 与NeRF的图像形成模型统一
论文证明了3D高斯溅射的α混合与NeRF的体素渲染**图像形成模型完全一致**，NeRF的体素渲染公式为：
$$C = \sum_{i=1}^N T_i \left(1 - e^{-\sigma_i \delta_i}\right) c_i, \quad T_i = e^{-\sum_{j=1}^{i-1} \sigma_j \delta_j} \tag{1}$$
令$\alpha_i = 1 - e^{-\sigma_i \delta_i}$（体素不透明度），$T_i = \prod_{j=1}^{i-1} (1-\alpha_j)$（透射率），则NeRF公式简化为：
$$C = \sum_{i=1}^N T_i \alpha_i c_i \tag{2}$$
而3D高斯溅射的α混合公式为：
$$C = \sum_{i\in\mathcal{N}} c_i \alpha_i \prod_{j=1}^{i-1} (1-\alpha_j) \tag{3}$$
两者完全等价，区别仅在于：NeRF通过**随机射线采样**获取体素样本（计算昂贵、有噪声），而3D高斯通过**显式溅射**直接提供样本（计算高效、无噪声）。

## 五、实验与结果
论文在**Mip-NeRF360、Tanks&Temples、Deep Blending、NeRF-synthetic**四大数据集上验证，硬件为NVIDIA A6000 GPU，对比方法包括Mip-NeRF360（SOTA质量）、InstantNGP、Plenoxels（快速方法），评估指标为**PSNR、SSIM、LPIPS**（质量）、**训练时间、渲染FPS、显存**（效率）。

### 5.1 定量结果
核心对比结果如表1所示，关键结论：
1. **Ours-7K（~5-10min训练）**：质量与InstantNGP/Plenoxels相当，渲染速度达134-197fps，远超对比方法；
2. **Ours-30K（~35-45min训练）**：质量与Mip-NeRF360（48h训练）持平/略优（如Deep Blending的PSNR 29.41 vs 29.40），渲染速度达137-154fps，是Mip-NeRF360的千倍以上；
3. **合成数据集**：随机初始化（无需SfM点云）即可达到SOTA，PSNR平均33.32，渲染速度180-300fps；
4. **紧凑性**：相比点基方法Zhang et al. 2022，用1/4的点数量实现更高PSNR，模型大小仅3.8MB（对比9MB）。

### 5.2 定性结果
1. 相比Mip-NeRF360，该方法消除了植被/墙面的模糊伪影，远距离视图仍保留细节；
2. 相比InstantNGP/Plenoxels，该方法的几何细节更丰富，无过/欠重建伪影；
3. 各向异性高斯能紧凑表示精细结构（如毛发、栏杆），而各向同性高斯会产生模糊。

### 5.3 消融实验
通过控制变量验证核心模块的有效性，定量结果如表3，关键结论：
1. **SfM初始化**：比随机初始化质量更高，减少背景浮点数高斯；
2. **各向异性协方差**：是质量提升的核心，各向同性高斯的PSNR平均低0.82；
3. **克隆+分裂**：缺一不可，分裂提升背景重建，克隆加速精细结构收敛；
4. **无溅射数量限制**：限制梯度传播的溅射数量会导致PSNR骤降（如Truck场景降11dB），视觉伪影严重；
5. **球谐系数（SH）**：提升视角相关颜色的表示，PSNR平均提升0.7。

## 六、局限性与未来工作
### 6.1 局限性
1. **未充分观测区域**：仍存在伪影（所有辐射场方法均有此问题）；
2. **弹出伪影**：大高斯的视锥剔除/深度排序切换会导致偶尔的弹出，源于简单的可见性算法；
3. **显存消耗**：训练大型场景时峰值显存超20GB（未优化的PyTorch实现），渲染显存为数百MB；
4. **超大型场景**：需降低位置学习率才能收敛，无正则化策略。

### 6.2 未来工作
1. 加入抗锯齿和更鲁棒的可见性算法，消除弹出伪影；
2. 为优化加入正则化，提升未观测区域的鲁棒性；
3. 将优化逻辑全移植到CUDA，降低显存消耗、提升训练速度；
4. 基于3D高斯实现网格重建，衔接体素表示和表面表示；
5. 结合点云压缩技术，进一步压缩模型大小。

## 七、核心结论
1. 3D高斯溅射是**首个实现辐射场高质量实时渲染**的方法，在1080p分辨率下达到≥30fps，训练时间35-45min，质量媲美Mip-NeRF360（48h训练）；
2. **各向异性3D高斯**是优秀的辐射场表示，融合了显式表示的快速光栅化和隐式表示的优化灵活性，无需MVS点云，仅SfM稀疏点即可初始化；
3. **瓦片式可微渲染器**是效率核心，通过GPU基数排序、瓦片级α混合实现实时渲染，无溅射数量限制的梯度传播保证了优化质量；
4. 实验证明：**连续的隐式表示并非辐射场高质量优化的必要条件**，显式的3D高斯表示在保持优化能力的同时，能实现数量级的速度提升。
